### 🧠 Overview
**Office Deployment Automation Suite (ODAS)** is a professional-grade framework for IT Administrators to manage Microsoft Office deployments across large-scale enterprise infrastructures. This toolkit leverages the Microsoft Office Deployment Tool (ODT) to provide a modular, repeatable, and scalable method for deploying, configuring, and updating Office installations without manual intervention.

### 🚀 Key Features
* **🛠️ XML Config Generator:** Build complex deployment configurations with a simple CLI interface.
* **⚡ Silent Deployment:** Automated, background installation pipelines for large-scale environments.
* **🧩 Modular Components:** Select specific apps (Word, Excel, Outlook) for targeted deployments.
* **🛡️ Compliance & Policy:** Enforce enterprise-wide settings and update channels during installation.
* **📊 Deployment Analytics:** Native logging to track success/failure rates across target machines.

### 📦 Installation
1. Ensure your environment has **PowerShell 7+** or **Python 3.12+**.
2. Download the `ODAS_v26_Installer.msi` from the latest release.
